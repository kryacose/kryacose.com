AREd: Engineering Graphics

An android app to make learning Engineering Graphics interactive by showing you the 3D object in AR. You can make your own models in app using the sandbox mode, or visualize the answers to graphics problems by superimposing the objects onto the drawings using the textbook mode. Built using Unity and Vuforia.

Github

Traffic App

I tried to recreate the live traffic feature in Google Maps by collecting accelerometer and gyroscope data from smartphones and using it to identify road conditions, pedestrian traffic and vehicular traffic patterns. This data is then superimposed onto the map to make it meaningful to the user. Built using Flutter.
